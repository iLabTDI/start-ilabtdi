## ¿Qué es?

La plantilla oficial del **iLab TDI** para proyectos web. Trae listo lo que normalmente se arma desde cero:

- Landing pública + **login / registro / reset** funcional
- Conexión a **Supabase** o a **MySQL** del hosting (dos caminos)
- **Deploy automático** a GoDaddy vía GitHub Actions
- Seguridad base (CSP, HSTS, RLS, validación con Zod)
- UI con Tailwind v4 + shadcn/ui, dark mode, componentes accesibles

## Filosofía

1. **Decisiones tomadas** — no te haces preguntas básicas cada vez (qué lib de forms, qué auth, dónde deployar).
2. **Sin features infladas** — trae lo indispensable. Tú agregas lo tuyo encima.
3. **Documentación viva** — si cambias una convención, actualiza la guía en el mismo PR.

## ¿Qué NO es?

- No es un framework — puedes ir contra las convenciones si tienes razón.
- No es todo-en-uno — si tu proyecto es una app móvil o un backend puro, este template no aplica.
